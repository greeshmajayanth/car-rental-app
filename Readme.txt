To run Backend:

Prerequistes- Python, visual build tools with C++

1. Connect to UTA VPN on your system using pulse secure VPN
Follow instructions in the below link to connect to UTA VPN: 
https://libguides.uta.edu/offcampus/vpn

2. Once you are in the backend folder, enter virtual environment using command- venv\Scripts\activate
3. Install necessary packages
4. Run main.py using command- python main.py

This should run the server on port 5000

To run Frontend:

Prerequistes- NodeJS 

1. Once you are in the frontend folder, install the node modules using command- npm install
2. Run the app using command- npm start

This will run the application on port 3000
